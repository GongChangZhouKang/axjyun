# Equipment Archive Source-Exact Design

本文件只服务于装备档案管理还原页，不作为通用主题。

## 视觉事实源
1. 源截图。
2. 导出 DOM 边界框和 computed styles。
3. 导出 `theme.json`。
4. Element Plus 默认样式仅作行为兜底。

## Tokens
- Primary: `#129BFF`
- Link / accent: `#2D8CF0`
- Ink: `#515A6E`
- Secondary ink: `#666666`
- Muted ink: `#808695`
- Border: `#DCDEE2`
- Canvas: `#F5F7F9`
- Filter surface: `#F5F7FB`
- Table head / zebra row: `#F4F5F7`
- Selected navigation: `#F0FAFF`
- Danger: `#F52C2C`
- Radius: `3px` / `4px`
- Body type: 14px / 21px, Helvetica Neue, PingFang SC, Microsoft YaHei, Arial, sans-serif

## Layout
- Fixed sidebar: 200px.
- Header: 64px.
- Tabs: 48px.
- Main content desktop inset: 20px.
- Inputs and buttons: about 34px high.
- Table header: about 38px; body rows: about 48px.
- On narrow screens the shell keeps its desktop minimum width; the viewport crops horizontally, matching the supplied screenshots.
