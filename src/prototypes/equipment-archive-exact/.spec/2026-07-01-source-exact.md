# 装备档案管理高精度还原确认快照

## 产品需求
- 目标用户：安保平台装备管理员。
- 核心任务：查询装备档案、维护装备、管理分类、切换启用状态并分页浏览。
- 页面范围：平台框架、装备管理侧栏、顶部页签、筛选区、工具栏、装备表格、分页和相关操作弹窗。
- 数据与内容来源：`AI-装备档案管理-complete.zip` 内截图、DOM、样式、主题令牌和文案。
- 实施边界：前端本地模拟交互，不接真实后端；不开发其他侧栏模块的完整页面。
- 验收重点：优先匹配 1912/1440 桌面截图，同时复刻 768/390 截图中的固定侧栏和裁切表现。

## 设计决策
- 不采用项目现有通用主题，建立页面专属 `DESIGN.md`。
- 事实源优先级：截图 > DOM/样式数据 > 导出主题令牌 > Element 默认样式。
- 固定 200px 左侧导航、64px 顶部栏、48px 页签栏和高密度表格。
- 使用源页面的亮蓝主操作、冷灰文字、浅灰筛选区、灰白斑马纹和紧凑 3–4px 圆角。
- Vue 3 + Element Plus 负责页面与交互，外观通过页面局部 CSS 精确覆盖。
