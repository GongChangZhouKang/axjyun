# 库存盘点确认快照

## 产品需求
- 用户：仓库管理员。
- 核心任务：新增盘点、开始盘点并锁库、录入实际库存、提交差异并解除锁库。
- 页面：盘点列表、新增/编辑盘点、盘点录入。
- 状态：待开始、进行中、已完成；待开始可编辑和删除。
- 单号：`PD + YYYYMMDD + 4位流水`。
- 锁定仓库仍可申请和创建出入库单，但不能执行实际出入库；停用状态仅说明、不在本页实现。

### 锁库业务阻断时机
- 调拨管理：新增调拨页点击“确认”时校验；调出仓库或调入仓库任一锁定即阻断。
- 出库管理：列表点击“出库”时阻断；新增出库页点击“确认”时阻断。
- 入库管理：列表点击“入库”时阻断；新增入库页点击“确认”时阻断。
- 锁库不阻断进入新增页面和填写数据，统一在上述关键动作执行前校验并提示。
- 阻断弹窗标题为“仓库正在盘点”，提示“该仓库正在盘点，暂不能进行库存操作。请在盘点完成后重试。”
- 弹窗展示盘点单号、盘点名称、盘点人、盘点仓库、盘点日期和盘点状态，仅提供“我知道了”按钮。
- 调拨业务中调出、调入仓库均被锁定时，分别列出两个仓库对应的进行中盘点信息。
- 阻断弹窗严格复用装备档案管理现有 `source-dialog` / `warning-dialog`：500px 居中弹窗、52px 灰色标题栏、双列浅灰信息块、56px 底部操作栏和右侧主按钮；不采用独立警告视觉。

## 设计决策
- 采用当前目录 `DESIGN.md`，继承平台壳、蓝灰色 Element Plus 组件与密集表格。
- 新增/编辑使用独立页面，不使用弹窗。
- 录入页实时显示差异，支持暂存、筛选、一物一码入口和提交确认。
- 操作文案：新增盘点、开始盘点、录入盘点、查看详情、保存盘点单、提交盘点。

## 实施边界
- 实现可运行前端原型及交互演示，不接真实后台、审批流和库存记账服务。
