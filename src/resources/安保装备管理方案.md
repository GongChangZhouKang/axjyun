## 一、管理依据
为规范公司保安员装备的采购、配备、申领、使用、维护、盘点和报废管理，确保装备配备合理、使用规范、责任明确、账实相符、流转可控，结合公司项目实际，制定本方案。

## 二、适用范围
本方案适用于公司集团、各分公司、项目驻点及保安员在在门卫、巡逻、守护、秩序维护、安全检查等保安服务活动中配备和使用的各类装备。

## 三、装备分类
结合行业标准和公司实际，保安员装备分为以下大类：

| 装备类别 | 主要内容 |
| --- | --- |
| **保安员服装** | 外衣裤、服饰、腰带、鞋、帽等 |
| **执勤装备** | 交通工具、照明设备、安检仪器、巡更设备、录音录像设备、通讯设备等 |
| **被动防护装备** | 防刺服、防暴头盔、防弹头盔、防弹衣、防爆毯、盾牌、防割手套等 |
| **主动防卫装备** | 防暴枪、保安防卫棍、约束叉等 |
| **消防装备** | 灭火器、防火毯、防火面具等 |
| **安检装备** | 通过式金属探测门、微剂量X射线安全检查仪、安检设备箱等 |


参考标准：**GA/T 1279-2015《保安员装备配备与管理要求》**

## 四、管理原则
1. **统一管理，分级负责**  
公司统一制定装备管理制度，各项目负责本项目装备日常管理。
2. **按岗配备，满足需要**  
根据岗位性质、服务内容、现场环境和风险等级合理配备装备。
3. **专人负责，账实一致**  
重要装备实行编号管理，明确保管人、使用人和管理责任。
4. **谁领用，谁负责**  
装备领用后由领用人或所在项目负责人负责保管、使用和归还。  
5. **规范使用，安全可控**  
装备报废、遗失、损坏须按流程审批，禁止私自转让、外借、变卖或丢弃。  
6. **定期检查，及时维护**  
对重点装备定期检查、维护、保养和更新，确保状态良好。

## 七、采购与入库管理
1. 项目或部门根据实际需求提出装备采购申请。
2. 公司管理部门审核装备类别、数量、用途和配置必要性。
3. 采购装备应选择质量可靠、资质合规的供应商。
4. 装备到货后应进行验收，核对名称、规格、数量、质量证明、使用说明等资料。
5. 验收合格后进行编号、登记、入库。
6. 对主动防卫装备、防暴类装备、安检类装备等重点装备，应加强采购、验收和备案管理

## 八、领用与发放管理
1. 装备领用实行申请审批和登记制度。
2. 使用人或项目填写领用申请，说明用途、岗位、数量和使用期限。
3. 项目负责人或公司管理部门审批后，由装备管理员发放。
4. 领用人应现场核对装备数量、编号和状态，并签字确认。
5. 临时借用装备应登记借用时间、归还时间和责任人。
6. 离岗、调岗、离职或项目结束时，应及时交回装备。

## 九、使用与保管管理
1. 装备仅限保安服务工作使用，不得用于与岗位职责无关的事项。  
2. 使用人员应按照说明和操作规范使用装备。 
3. 装备应明确使用对象，指定专人管理 ，严格控制使用范围。  
4. 重点装备应妥善保管，防止遗失、损坏或被他人违规使用。  
5. 不得使用已损坏、失效或超过使用期限的装备。
6. 装备使用人员离职、调岗或岗位服务结束时，应及时交回所使用或保管的装备，保安服（不含标志）除外。

## 十、检查、维护与盘点
1. 使用人应在交接班时检查装备数量和状态。
2. 班组长负责每日或每班次清点检查。
3. 项目负责人每周组织项目装备检查。
4. 公司管理部门每月或每季度组织抽查和盘点。
5. 对防刺服、防弹衣、防暴头盔、防暴枪、防卫棍、约束叉、安检设备、对讲机、录像设备等重点装备，应建立检查维护记录。
6. 发现装备损坏、失效、缺失或超过使用期限的，应及时维修、更换或报废。

## 十一、损坏、遗失与报废管理
1. 装备损坏、遗失后，使用人应立即报告项目负责人。
2. 项目负责人核实情况后填写损坏或遗失说明。
3. 管理部门根据原因和责任情况，组织维修、更换、赔偿或处理。
4. 装备达到使用年限、无法维修、性能失效或存在安全隐患的，应按程序申请报废。
5. 报废装备经审批后统一处置，并从台账中注销。
6. 重点装备的报废和处置应符合监管要求，严禁私自处理。
