# 安保装备管理主题

## 1. 视觉主题与氛围

面向安保装备、库存、盘点和出入库等高频后台作业。界面应专业、克制、可靠，以高信息密度和快速扫描为首要目标。品牌表达集中在顶部平台栏与主色，不使用营销式大标题、渐变背景或大面积装饰。

- 适用：装备档案、库存台账、盘点任务、出入库单据、分类配置。
- 不适用：移动端领用、营销官网、沉浸式数据大屏。
- 关键词：清晰、稳定、紧凑、可追溯、操作优先。
- 架构原则：平台框架只实例化一次；业务页面只提供内容区，独立预览时才组合完整框架。

## 2. 色彩系统

- `primary #129BFF`：主按钮、当前导航、焦点环、关键状态；禁止用于大面积背景。
- `link #2D8CF0`：文字链接、活动页签和可交互文本。
- `canvas #F5F7F9`：页签轨道与页面分区背景。
- `surface #FFFFFF`：页头、侧栏、表格、表单和弹窗表面。
- `surface-subtle #F5F7FB`：筛选区；同一页面不叠加多层灰底。
- `table-head #F4F5F7`：表头和斑马纹行。
- `ink #515A6E`：正文和常规控件文字。
- `ink-strong #30384B`：页面标题、区块标题。
- `muted #808695`：说明、次要信息和占位提示。
- `border #DCDEE2`：控件、分隔线和表格边界。
- `success #52C41A`、`warning #F5A623`、`danger #F52C2C`：只表达业务状态和风险操作。
- 选中导航使用 `#F0FAFF`，表格悬停使用 `#EBF7FF`。

## 3. 字体系统

- Display / Body：`Helvetica Neue`, `PingFang SC`, `Microsoft YaHei`, Arial, sans-serif。
- Mono：Consolas, `Liberation Mono`, monospace，用于装备编码、单号和序列号。
- 页面标题：18px/24px，700；区块标题：16px/24px，600。
- 正文与控件：14px/21px，400；表头：14px/20px，700。
- 辅助说明：12px/18px，400。
- 数字列启用等宽数字；禁止用轻字重承载关键数据。

## 4. 组件规范

- 顶部栏：64px，高亮品牌区域，白底，底部轻阴影。
- 侧边栏：桌面宽 203px；一级和二级导航均高 49px；选中项浅蓝底并在右侧显示 2px 主色标记。
- 页签：轨道高 48px，活动页签白底、蓝色文字，与内容区连通。
- 按钮：默认高 32px，圆角 4px；主按钮蓝底白字；危险文字按钮只用于删除、作废。
- 输入与选择器：高 34px，圆角 4px，1px 边框；聚焦使用主色边框与浅蓝焦点环。
- 筛选区：浅灰蓝表面，圆角 6px，16px 内边距；桌面默认三列；查询与重置靠右。
- 表格：表头 38px，数据行 48px；允许斑马纹；行悬停浅蓝；操作列固定在右侧。
- 状态标签：使用状态色圆点或轻量标签，文字必须同时说明状态，不单独依赖颜色。
- 弹窗：标题栏 52px、底部操作栏 56px、圆角 8px；确认操作在右。
- 表单页：区块标题左侧 6px 主色短条；长流程使用底部固定操作栏。
- 分页：34px 控件，靠右排列，当前页用主色边框和文字。

## 5. 布局与间距

- 桌面框架：顶部栏 64px，侧边栏 203px，页签 48px，内容区内边距 20px。
- 内容最大宽度不强制锁定，数据表格优先利用可用横向空间。
- 间距标尺：4、8、12、16、20、24、32、40px。
- 筛选项间距 12px；工具栏高度 56px；区块间距 24–32px。
- 框架组件与业务页面组件必须分离。路由切换只替换内容区，不重新挂载顶部栏和侧栏。

## 6. 深度、阴影与边框

- 常规控件和表格使用 `1px solid #DCDEE2`。
- 顶部栏：`0 1px 6px rgba(0,0,0,.16)`。
- 固定底栏：`0 -2px 7px rgba(0,0,0,.05)`。
- 弹窗：`0 12px 32px rgba(23,35,61,.16)`。
- 卡片默认无阴影，以背景和边框分层；禁止厚重投影、发光边框和玻璃拟态。

## 7. 动效

- 悬停与聚焦：120–160ms `ease-out`，只改变颜色、边框或轻微背景。
- 抽屉与弹窗：180–220ms `cubic-bezier(.2,.8,.2,1)`。
- 开关滑块：200ms `ease`。
- 不使用弹跳、连续漂浮、复杂视差或影响表格扫描的行入场动画。
- 尊重 `prefers-reduced-motion`，关闭非必要位移动效。

## 8. 响应式行为

- `>= 1280px`：完整侧栏，筛选区三列，操作列固定。
- `960–1279px`：侧栏可折叠为图标栏，筛选区两列，表格横向滚动。
- `< 960px`：侧栏改为抽屉，页签横向滚动，筛选区单列；主操作保留，次要操作收进“更多”。
- 主题本身不定义移动端装备领用界面；移动端业务另用专用主题或页面规范。

## 9. Prompt guide

推荐写法：

- “使用安保装备管理主题生成装备档案内容区，复用现有平台框架。”
- “使用三列紧凑筛选区、38px 表头和 48px 数据行，操作列固定在右侧。”
- “用状态文字配合颜色圆点表达启用、停用、待盘点等状态。”
- “新建库存页面时只生成业务内容，不重复顶部栏、侧栏和页签容器。”

禁止写法：

- 不生成营销式 Hero、大渐变或大圆角卡片墙。
- 不为每个页面重复生成完整平台框架。
- 不使用只有图标、没有文字或提示的关键业务操作。
- 不把移动端领用组件混入桌面后台主题。

可复用提示：

> 基于安保装备管理主题创建生产级后台页面。保持白色表面、浅灰蓝筛选区、蓝色关键操作和紧凑数据表格。若已有平台框架，只输出业务内容区；仅在独立预览时组合顶部栏、侧边栏和页签。所有状态同时使用文字与颜色，复杂表格在窄屏允许横向滚动。
