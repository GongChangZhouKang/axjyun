# 一物一码连续录入与盘点核对

## 目标
在 `equipment-archive-exact` 中新增独立连续录码页与盘点物码核对页，完整实现扫码入列、库内匹配、待建档、冲突阻断和盘点数量回写。

## 已确认决策
- 视觉严格沿用现有 `DESIGN.md` 与用户截图中的蓝灰色 Element 风格。
- Page 1 不展示“添加方式”列。
- Page 2 使用单列表关系单元格，不采用左右两栏。
- 未知码暂存为待建档并计入实盘；提交整张盘点单前统一确认建档。
- 已有但不属于当前装备、当前仓库或非在库状态的物码直接阻断，不允许强制改绑。
- 使用 Edge 完成交互与视觉验收。
