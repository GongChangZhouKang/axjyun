# 移动端库存盘点

## 目标
在 `equipment-archive-exact` 原型内新增符合现有 APP 截图风格的移动端库存盘点闭环，并保留 Axhub 原型标注。

## 已确认需求与设计
- 用户：仓库管理员、盘点经办人。
- 核心路径：功能中心入口 → 盘点列表 → 新增/开始 → 数量录入或扫码 → 保存 → 提交。
- 状态：待开始、进行中、已完成；开始后锁库，提交后解除锁库。
- 数据：与现有后台盘点示例数据共用；不实现真实接口、审批和差异单处理。
- 视觉：移动端页面参考用户提供的现有 APP 截图；颜色与组件语言继承原型现有 `DESIGN.md`。
- 弹窗：严格复用装备档案管理的标题、圆角、遮罩、间距、分隔线、右对齐取消/确认按钮规范，只做移动端宽度适配。
