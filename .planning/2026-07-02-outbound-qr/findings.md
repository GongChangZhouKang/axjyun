# Findings

- 用户提供的原系统截图确定了新增出库页的信息区、明细表格、底部操作栏及按钮位置。
- 用户手绘二维码稿确定了下载图片的标题、扫码路径、二维码、装备清单和安全提示结构。
- 当前原型为 React 接入层 + Vue/Element Plus 页面，标注已通过 `annotation-source.json` 静态接入。
- 当前设计基底是原型目录内的 `DESIGN.md`，使用 200px 侧栏、64px 顶栏、48px 标签栏与蓝灰 Element Plus 风格。
- 项目现有依赖中包含二维码编码实现，可在不新增 package 依赖的情况下生成可识别矩阵并绘制到下载图片。
- 目标页在 1920×950 视口下与参考截图布局一致；标注工具会显示彩色 marker，普通预览不影响页面本身。
